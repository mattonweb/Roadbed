# Roadbed.IO

File I/O utilities with strongly-typed wrappers.

## Overview

This library provides type-safe file operations. It includes a simplified file info wrapper.

## Installation
```bash
dotnet add package Roadbed.IO
```

## Requirements

- .NET 10.0+

## Related Packages

- **Roadbed.Crud** - CRUD patterns for entities and repositories
- **Roadbed.IO.Csv** - CSV implementation and file type entities
